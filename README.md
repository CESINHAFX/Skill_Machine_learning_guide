# Skill_Machine_learning_guide
SkillGuide-MachineLearning é um repositório de apoio ao Copilot, voltado para orientar e estruturar projetos de aprendizado de máquina. Ele reúne instruções, exemplos e boas práticas para todo o ciclo de desenvolvimento de modelos, desde a preparação de dados até a avaliação e implantação.
O conteúdo está organizado em diretórios e documentos específicos, permitindo que o Copilot utilize as skills de Machine Learning de forma contextualizada durante a execução de scripts como sdlc_guide.py
